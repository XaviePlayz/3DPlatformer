# 3DPlatformer

Game Goal:
Eliminate All Jesters.

Game Mechanics:
Move - WASD.
Sprint - Ctrl.
Look - Mouse.
Shoot - Left Mouse Button.
Zoom - Right Mouse Button.
Pick-up - F.
Use - E.
Drop - Q.
Rotate Spells - 1, 2 & 3.
